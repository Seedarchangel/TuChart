## 数据类型
Tuchart目前支持的数据类型有：
1. *未复权日线
  周线
  月线
  15分钟线
  30分钟线*
2. *前／后复权日线，周线，月线，5分钟线，15分钟线,30分钟线*
3. *个股分笔交易数据*

## 依赖
请安装anaconda 2.7,并另外安装

1. pyecharts
2. tushare

使用```pip install pyecharts, tushare```即可安装

## 使用方法
```python main.py```
